# BlockViz Public

Public, non-sensitive artifacts for **BlockViz** — blockchain data visualization.

- Website: https://blockviz.xyz

This repository contains documentation, lightweight data contracts, and sample payloads used to describe BlockViz outputs.
Core BlockViz systems and services are proprietary and **not** included here.

## What’s inside

- `api/` — JSON Schema definitions for public payload shapes
- `examples/` — small example payloads (synthetic)
- `docs/` — branding and data contract notes
